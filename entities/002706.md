| 维度 | 详情 |
|------|------|
| **SST必要性** | SST输出侧多模块并联，故障需微秒级切除（SSCB几十微秒 vs 机械几十毫秒），**SSCB是SST必选项** |
| **价值量** | 单个800V模块配2个SSCB，单价5000元；2MW-SST单机27万元，折1.35亿/GW |
| **价格对比** | 固态微断**4000-5000元** vs 传统几十元（价差20-30倍） |
| **空间测算** | 2030年仅美国：50GW×SST渗透40%×7亿/GW = **140亿市场，利润空间56亿** |
| **AIDC** | 与维谛、台达联合开发北美数据中心全套低压配电，26Q2末上市，26H2批量 |
| **估值路径** | 26-28E净利 2.99→4.41→**7.95亿**(+13%/48%/80%)，28E PE 19x |
