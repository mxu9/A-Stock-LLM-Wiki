# 航天工程 (603698.SH)

## 📌 主营业务

航天粉煤加压气化技术（航天炉）为核心，专业从事煤化工气化炉EPC总承包。航天炉市占率>50%（粉煤气化领域），获"制造业单项冠军"。第二曲线：绿氢电解槽 + 生物质气化 + 高端阀门。

> 🔥 **航天一院（中国运载火箭技术研究院）旗下唯一A股上市平台**，控股45.98%，集团合计~67%。
