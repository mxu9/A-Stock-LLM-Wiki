## 五、与第一梯队（设备）的联动逻辑

```
长鑫295亿IPO募资到位
        │
  ┌─────┴─────┐
  │ 设备招标启动 │  ← 第一梯队受益（刻蚀/薄膜/CMP/清洗/量测）
  └─────┬─────┘
        │ 订单落地 → 设备厂批量采购零部件
  ┌─────┴─────┐
  │ 零部件放量  │  ← 第二梯队受益（腔体/阀门/射频电源/石英件/硅电极）
  └─────┬─────┘
        │ 产线爬坡 → 耗材持续消耗
  ┌─────┴─────┐
  │ 材料耗材放量 │  ← 第三梯队受益（硅片/特气/光刻胶/抛光液）
  └───────────┘
```

**第二梯队的独特优势**：
- **弹性更大**：零部件企业营收基数普遍小于设备龙头，同等订单增速对应更高利润弹性
- **国产化率更低**：珂玛/神工国产化率<5%，替代空间远大于设备（设备国产化率~24%）
- **耗材属性**：珂玛的加热盘/ESC、神工的硅电极具有耗材复购特性，产能爬坡后持续受益

## 六、风险提示

1. **订单传导滞后**：设备订单→零部件采购存在3-6个月滞后期，短期业绩兑现可能不及预期
2. **产能爬坡风险**：富创北京基地利用率仅16.82%，若爬坡不及预期将拖累毛利
3. **大股东减持**：正帆科技控股股东拟减持0.95%（7/30-10/29）
4. **竞争格局恶化**：先锋精科/恒运昌/中科仪等非上市竞品同步扩产
5. **技术壁垒**：高端阀门/射频电源等核心品类仍被海外垄断，国产突破需时间验证
