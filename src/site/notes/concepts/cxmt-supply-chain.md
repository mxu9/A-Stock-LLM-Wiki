# 长鑫存储产业链

> 🚀 7/16 IPO申购(295亿) | H1预盈500-570亿(+2244%) | 全球DRAM市占率8%→17%(2028E) | 43家A股产业链标的

## 概览

长鑫科技（CXMT）是中国最大、全球第四大DRAM制造商（IDM全产业链模式）。2026年7月16日科创板IPO申购，募资295亿创A股今年最大IPO。全年计划扩产5-6万片/月，设备采购350-430亿元。产业链覆盖设备→零部件→材料→封测→模组全环节。

## 📊 机构观点

| 机构 | 核心判断 |
|------|----------|
| **国联民生** | 设备先行：刻蚀/薄膜沉积/CMP/清洗直接受益；零部件弹性最大；材料后周期持续性强 |
| **东吴证券** | 国内存储扩产设备订单绝大部分由国产公司承接，刻蚀/薄膜沉积双重受益 |
| **西部证券** | AI驱动半导体设备上行周期，前道设备高景气+后道封测扩产 |
| **招商证券** | 全球DRAM供需缺口持续至2027+，合约价涨势延续 |
| **花旗** | 长鑫供应链中优先看好ASMPT及长电科技 |
