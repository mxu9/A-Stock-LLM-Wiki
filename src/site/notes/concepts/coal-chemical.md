# 🏭 煤化工

> 以煤为原料的化学加工工业，保障能源安全与化工产业链自主可控的战略产业 | 26年煤油比8.07处于历史低位 → 盈利上行周期

## 📊 行业概览

| 指标 | 数据 | 来源 |
|------|------|------|
| 市场规模(2026) | **突破1.2万亿元** | 中研普华(26/6) |
| 新项目投资 | **6000亿元**（在建+规划） | 天风证券(26/5) |
| 煤油比(2026) | 8.07（vs 2025的10.42 ↓22%） | 天风证券(26/6) |
| 动力煤消费增速 | Q1同比+12% | 北京研精毕智(26/4) |
| 行业利润率(Q1) | 12.3%（+3.5pct YoY） | 北京研精毕智(26/4) |

## 🔗 产业链

```
煤炭开采 → 煤气化 → 合成气(CO+H2) → 甲醇/合成氨
                                         ├── 烯烃(CTO) → PE/PP/EVA
                                         ├── 醋酸/DMF/DMC/己内酰胺
                                         ├── 尿素(化肥)
                                         └── 煤制油/煤制气
```

## 🔑 核心投资逻辑

1. **煤油比套利**：Brent $96 vs 秦港Q5500 775元，煤油比8.07 → 煤化工成本优势历史性放大
2. **产能释放**：6000亿新项目集中于2026-27投产，龙头企业增量明确
3. **新材料升级**：EVA/POE/DMC/EMC/TDI高附加值转型打开第二曲线
4. **估值洼地**：华鲁PE 14.8x/PEG 0.36，广信PB 0.90破净

## ⚠️ 风险

- 碳排放双控政策压制长期估值
- 6000亿新投资带来3年内供给冲击
- 煤价上行风险（775→850元）
- 精细化工面临海外竞争
