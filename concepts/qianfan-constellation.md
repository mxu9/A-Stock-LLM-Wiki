# 千帆星座（G60星链）

> 中国首个进入正式组网阶段的巨型低轨商业卫星星座。对标 SpaceX Starlink，由上海垣信卫星运营，规划 >1.5万颗卫星，目标市场"一带一路"+中资企业出海。

## 核心数据

| 维度 | 详情 |
|------|------|
| **运营方** | 上海垣信卫星科技有限公司（上海国资主导） |
| **总规划** | **>15,000颗**，三期建设 |
| **当前在轨** | **200颗**（2026年6月） |
| **发射节奏** | 月均18颗，一箭18星（平板卫星堆叠发射） |
| **一期目标** | 1296颗（2027年完成），区域覆盖 |
| **二期目标** | ~10,000颗（2030年），全球覆盖 |
| **三期终态** | >15,000颗，融入6G空天地一体网 |
| **2026目标** | 发射540颗，在轨324颗，年底启商 |
| **融资** | A轮67亿(2024) + **新一轮≥50亿(2026.6，拒绝境外资本)**，估值750亿 |
| **里程碑** | 2026.6 全球首次存量手机无需改造直连卫星通话（200Mbps） |
| **运营变动** | **2026.3 总裁沈洪波(原上海联通总经理)就任，微系统所背景CTO** |

## 技术特点

| 维度 | 详情 |
|------|------|
| 卫星构型 | 平板卫星+堆叠发射，一箭18星 |
| 单星成本 | 一代星~3000万，手机直连星<950万（较国际低68%） |
| 星间链路 | 100G-400G激光通信 |
| 手机直连 | 碳化硅基板射频模块，发射效率58% |
| 信关站 | 28个区域信关站，单站10Gbps |
