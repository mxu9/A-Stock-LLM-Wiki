# 氦气板块

> "气体稀土" — 半导体/航天/医疗不可替代的战略稀有气体。中国84%依赖进口，2026年供给冲击触发国产替代加速。

## 📊 市场概况

| 指标 | 数值 |
|------|------|
| 全球氦气资源总量 | ~519亿m³ (USGS) |
| 中国资源量 | ~11亿m³ (全球占比2.12%) |
| 中国进口依赖度 | 84.44% (2025, 隆众资讯) |
| 国产管束高纯氦 | 76→153元/m³ (+132%, 2026.2-7) |
| 核心下游 | 半导体EUV(35%) + 航天(20%) + MRI(18%) |

## 🔗 产业链

```
气源(提氦) → 纯化(5N→6N→7N) → 设备/储运 → 分销/回收
```
